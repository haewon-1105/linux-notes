# Day 1 - Linux Basics

## What I Learned

On Day1 I learned the basics of Linux terminal commands.

---

## Users and Groups

### whoami

Shows the current logged-in user.

Example:

```bash
whoami
```

### id

Shows user ID (UID), group ID (GID), and groups.

Example:

```bash
id
```

### id -un

Shows only the username.

Example:

```bash
id -un
```

---

## File and Directory Management

### pwd

Shows the current directory.

### cd

Changes directory.

### touch

Creates a file.

### mkdir

Creates a directory.

### ls

Lists files and directories.

### cp

Copies files and directories.

### mv

Moves or renames files.

### rm

Deletes files.

### rmdir

Deletes empty directories.

---

## Viewing File Contents

### cat

Displays the contents of a file.

### cat -n

Displays file contents with line numbers.

### head

Displays the beginning of a file.

### tail

Displays the end of a file.

### diff

Compares files.

### diff -r

Compares directories recursively.

---

## Important Notes

- Linux uses users and groups to manage permissions.
- rm permanently deletes files, so use it carefully.(especially -f)
- cat, head, tail, and diff are useful for inspecting files.
- Practicing commands regularly is important for building Linux skills.
